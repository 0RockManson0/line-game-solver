# Line Game Solver (Игра "Шарики")

Программа моделирует механику удаления шариков в линии. Если образуется непрерывная цепочка из 3 и более шариков одного цвета, она удаляется, а остальные шарики сдвигаются друг к другу. Процесс повторяется до тех пор, пока новые цепочки перестанут образовываться.

## Как работает алгоритм
1. Программа получает список цветов шариков.
2. Алгоритм проходит по списку слева направо и ищет первую группу из 3 и более одинаковых чисел подряд.
3. Если группа найдена, она удаляется из списка. Оставшиеся элементы автоматически сдвигаются, заполняя образовавшуюся пустоту.
4. После сдвига проверка списка начинается заново, так как могла сформироваться новая цепочка.
5. Цикл завершается, когда за полный проход не найдено ни одной цепочки. Программа выводит пошаговый отчет и общее количество уничтоженных шариков.

4. Запустите программу командой:

   python main.py
